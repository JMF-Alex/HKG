## About the Project

**HKG** is A simple keylogger made with Python and tkinter

### Features

- Export to TXT files
- Start and stop easily

## Built With

This project was developed using:

- **Python** for backend logic
- **Tkinter** for Python GUI applications

Repository: [github.com/JMF-Alex/HKG](https://github.com/JMF-Alex/HKG)

## Getting Started

### Prerequisites

Check `requirements.txt` for the required dependencies.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/JMF-Alex/HKG.git
```

2. Enter the project directory:

```bash
cd HKG
```

3. Open `src/main.py` with your python interpreter.

That's it — the app runs entirely locally!
